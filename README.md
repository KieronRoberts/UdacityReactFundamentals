## Run Locally

Clone the project

```bash
  git clone https://github.com/KieronRoberts/UdacityReactFundamentals
```

Go to the project directory

```bash
  cd UdacityReactFundamentals
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
